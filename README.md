<h1 align="center">SupaClick!</h1>

SupaClick! is a casual web-based browser game designed to test players' speed and reflexes. Players must click or tap on various "Emojis" falling from the top of the screen to earn points. Each emoji type triggers a different outcome ranging from score boosts and penalties to an immediate "Game Over." The game’s difficulty scales dynamically as the player’s score increases.
> [!IMPORTANT]
> The game is currently under development.

# Objectives
* Accessibility & Engagement: To create a web game that is easy to pick up, challenging to master, and suitable for all ages.
* Cross-Platform Support: To ensure seamless gameplay across devices, supporting both mouse clicks (PC) and touch gestures (Mobile/Tablet).
* High Performance: To deliver a smooth experience using standard Web Technologies and Modern Frameworks instead of heavy, specialized game engines.
